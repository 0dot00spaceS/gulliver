# Guilliver

A PWA Directory example.

## Installation

1. Fork the repository.
2. Install [NodeJS](https://nodejs.org).
3. Setup the repository.

```none
$ git clone https://github.com/GoogleChrome/gulliver.git
$ cd gulliver
$ npm i
$ npm start
```
